About Us

Real Estate Registry Authority is an Act that serves as an intermediary between the user and government procedures and helps user to make Registration Process of their property easier.
This application is intended to help user in order to get the desired seeking information as well :
1. View list of registrations of different properties done so far by the user.
2. View details of particular property using a unique property number or id i.e. property value, property area, property location, last owner of property and so on.
